## 11. 错误处理与用户体验

### 11.1 错误分级

| 级别 | 颜色 | 示例 | 处理方式 |
|------|------|------|----------|
| **致命** | 🔴 红色 | CLI 未安装、进程崩溃 | 全屏提示 + 重试按钮 |
| **警告** | 🟡 黄色 | 网络超时、工具失败 | 内联提示 + 自动重试 |
| **信息** | 🔵 蓝色 | 会话恢复、状态变更 | 轻量提示 |

### 11.5 快捷键提示

| 操作 | 快捷键 | 提示位置 |
|------|--------|----------|
| 发送消息 | Enter | 输入框占位符 |
| 换行 | Shift+Enter | 输入框占位符 |
| Inline Edit | Ctrl/Cmd+Shift+E | 工具栏提示 |
| Plan Mode | Shift+Tab | 操作栏提示 |
| 取消 | Esc | 流式时显示 |
| 新标签页 | Ctrl/Cmd+T | 标签栏提示 |

### 11.7 性能优化

| 优化点 | 策略 |
|--------|------|
| 消息渲染 | 虚拟滚动（长会话） |
| 文件加载 | 延迟加载大文件 |
| 会话恢复 | 按需加载消息历史 |
| 工具调用 | 异步执行，不阻塞 UI |
| 内存管理 | 及时清理已完成的工具调用数据 |
